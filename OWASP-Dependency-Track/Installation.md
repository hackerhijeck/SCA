## OWASP Dependency Track

### Installation and Usage:
```
## Install Java 17 or above
$ apt update -y
$ apt install openjdk-17-jdk
$ wget https://github.com/DependencyTrack/dependency-track/releases/download/4.10.1/dependency-track-bundled.jar
$ java -Xmx8G -jar dependency-track-bundled.jar
## Default user port
http://localhost:8080
## Default admin API port
http://localhost:8081

## Default credentials:
admin:admin
```
